WordPress Moderator Tools

Tools for WordPress dot org forum moderators

Keyboard navigation. Version 0.1.

Note: This is not the latest version, check out the latest stable release.

A Javascript tool that adds keyboard shortcuts and a menu to the bbPress posts (support/bb-admin/posts.php) and edit profile pages (/support/profile/dot-org-profile/edit). See Usage.

This tool can be installed in three ways, as a Javascript bookmarklet, a Firefox add-on, or a Greasemonkey user script (Firefox, Chrome). The add-on and user scripts run automatically on the specific pages and the bookmarklet has to be clicked from the browser bookmark menu.

See the Disclaimer on how you can use this in other browsers.

Older versions can be found at the Release Archive or Github.

Javascript bookmarklet

Drag this bookmarklet to your browser bookmark menu: Moderate

Go to a posts (support/bb-admin/posts.php) page or a profile (/support/profile/dot-org-profile/edit) page and click the bookmarklet in your browser bookmark bar.

Firefox add-on

Install the Firefox add-on if you don't have Greasemonkey installed.

Click to install the Firefox Addon

Or download the Firefox Addon and add it manually.

You probably have to restart Firefox for the add-on to be activated

The add-on was created by this Greasemonkey script compiler.
Code used in the compiler: wp_moderator_shortcuts.js

Firefox Greasemonkey script

Install this user script if you have Greasemonkey installed.

Click to install: Firefox Greasemonkey user script

Chrome Greasemonkey script

(No need to install Greasemonkey)

  1. Click this Chrome Greasemonkey user script (Chrome will download the script, because it doesn't allow scripts to be added outside the Chrome Web Store)
  2. Go to "Window" > "Extensions"
  3. Drop the "wp_greasemonkey_Chrome.user.js" file in the "Extensions" page
  4. Click "Add"

Or download the script and drop the file in the Chrome "Extensions" page: wp_greasemonkey_Chrome.user.js

Usage

A "Edit Profile" link is automatically added to all posts.

You can select a post by clicking in the author column. The author column gets a green background color when the post is selected (for visibility).

By typing "m" you open the menu.

Clicking "shortcuts" or typing "h" displays all shortcuts available for that page.


The menu ("m") on a profile edit page (/support/profile/dot-org-profile/edit) only displays the shortcuts available:

Go and play and test all the shortcuts for yourself.

Defaults

The Greasemonkey user scripts hides the menu by default and uses a green color for selected posts. By editing the Greasemonkey file you can have it show the menu by default and change the selected color. Find var c="hide";var m="#d5e8d2"; at the start of the minified javascript. Change "hide" to show var c="show"; and change the color in var m="#d5e8d2"; to your liking and save the file.

Disclaimer

All the scripts are tested in Firefox and Chrome only. If other browsers support Greasemonkey user scripts you can try with one of the Greasemonkey users script js files found on this page. The Javascript bookmarklet can be used on all browsers (I think).
